# Real-Time Market Analytics Dashboard

A real-time market analytics dashboard that ingests live market data,
processes it in memory, performs analytics, and visualizes insights
through an interactive dashboard.

## Features
- Real-time market data ingestion
- WebSocket-based streaming
- In-memory data storage
- Analytics and resampling
- Interactive dashboard visualization

## Project Structure
analytics/   - analytics and resampling logic  
ingestion/  - live market data ingestion  
storage/    - in-memory data store  
frontend/   - dashboard UI  
app.py      - main application  

## System Flow
1. Market data is ingested in real time
2. Data is stored in memory
3. Analytics module processes data
4. Dashboard displays insights

## Tech Stack
- Python
- WebSockets
- In-memory data structures
- Git & GitHub

## How to Run
git clone https://github.com/AryaSawane/Real-Time-Market-Analytics-Dashboard.git  
cd Real-Time-Market-Analytics-Dashboard  
python app.py
